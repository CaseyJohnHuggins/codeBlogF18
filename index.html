Assignment 3

That feeling when you finally get around to hitting a programming assignment and you hit it hard and fast and knock it out in one sitting!

....Okay so that probably wasn't a good idea, but it worked out this time.

This assignment actually worked out really well for me because a lot of it was stuff I'd previously done for classes in the past.

The most helpful resource was, once again, the classroom. I can't emphasize that enough. I might look like I'm falling asleep in the middle of the afternoon lectures, but my notebook is packed with notes. I promise. Seriously. I only googled things as sanity checks this time around, or the occasional mixup like using ".add" instead of ".push" to append to an array. I'm actually starting to come around to Javascript. Before this class, web development was the devil to me.

I had some sweet plans to add some really cool bells and whistles to my code blog this week. That may or may not end up happening. I'll probably end up tweaking it when I'm supposed to be working on my next Data Structues assignment.
